# SoundScript

A lightweight, browser-based speech recognition tool that converts live audio into text in real-time. This project features instant clipboard integration and a clean, responsive UI.

## ✨ Features

* **Real-Time Transcription:** Uses the Web Speech API to turn your voice into text instantly.
* **One-Click Copy:** Automatically copies the generated text to your clipboard for easy pasting.
* **Manual Controls:** Includes **Stop** and **Delete** buttons to manage your recording session and clear the output.
* **Pure Web Tech:** Built with zero dependencies—just HTML, CSS, and Vanilla JavaScript.

## 🛠️ Built With

* **HTML5:** Semantic structure.
* **CSS3:** Modern styling and responsive layout.
* **JavaScript (ES6):** Logic for Speech Recognition and Clipboard API.

## 📋 How to Use

1.  **Start Speaking:** Most browsers will prompt for microphone access once the script initializes.
2.  **View Output:** Your speech will appear in the main text area.
3.  **Copy** * The output is manually copied.
    * Click **Stop** to end the recognition session.
    * Click **Delete** to wipe the current text and start fresh.

## ⚙️ Setup

To run this project locally:

1.  Clone the repository:
    ```bash
    git clone https://github.com/arijeetpandey/soundscript.git
    ```
2.  Navigate to the project folder:
    ```bash
    cd soundscript
    ```
3.  Open `index.html` in any modern web browser (Chrome or Edge recommended for best Speech API support).

## ⚠️ Requirements

* **Browser Support:** This project requires the [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API), which is best supported in Google Chrome and Microsoft Edge.
* **Microphone:** A working microphone is required for speech input.
